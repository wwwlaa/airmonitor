Sistem Monitoring Kualitas Udara
Dirancang untuk membantu sivitas akademika dan masyarakat memantau kualitas udara area indoor maupun outdoor secara real-time menggunakan teknologi IoT.

Kerja Praktik IoT

Project KP berbasis Internet of Things dengan implementasi sensor lingkungan dan web dashboard real-time.

Informatika Untan

Dirancang untuk membantu sivitas akademika memantau kualitas udara di ruang lab dan koridor Informatika Untan.

ESP8266 + Sensor

Hardware menggunakan NodeMCU ESP8266 dengan sensor MQ-135 (gas) dan DHT11 (suhu & kelembapan).
